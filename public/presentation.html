From CBS to PBS: Scaling MAPF Solvers

Conflict-Based Search (CBS)

Optimal Exponential

Branches on conflicts via spatiotemporal constraints. Optimal but scales to ~200 agents.

(Sharon et al., 2015 — Course week 7)

Priority-Based Search (PBS)

Suboptimal Scalable

Resolves conflicts via priority orderings. Lazy, dynamic. Handles hundreds of agents.

(Ma et al., 2019)

Key: PBS is not Prioritized Planning. PP fixes ordering upfront; PBS searches over orderings dynamically. PBS is the subproblem solver used in this paper.
Compare CBS (optimal, exponential) to PBS (suboptimal, scalable). PBS is the inner solver for the LNS framework.

Why 3D Conv?

  • Captures LOCAL spatiotemporal interactions — agent collisions, proximity in space-time
  • Preserves H x W x T tensor structure

Why Intra-Path Attention?

  • Conv is local — can't propagate info along a long winding path
  • Attention gathers features along each agent's trajectory → GLOBAL information flow
Key advantage: Heavy 3D conv + attention computed once (shared). Only lightweight heads run J times → ~2x faster than Per-Subset baseline.

Critical Analysis

Strengths

  • ✓Novel architecture — First deep learning approach for LNS guidance in MAPF. Opens new research direction.
  • ✓Principled design — Every component (3D conv, attention, shared backbone) has clear motivation.
  • ✓Runtime–quality gains — Achieves 1.5×–4× speedup over unguided LNS while maintaining or improving solution quality.
  • ✓Generalization capability — Performs well on unseen maps and agent configurations, showing the model captures transferable patterns rather than overfitting.

Weaknesses

  • ✗Limited autonomy — NN only ranks predefined neighborhoods instead of generating them, part of the decision process depends on heuristics.
  • ✗Inference overhead — Neural guidance must run at every LNS iteration; even fast inference (<0.05s) accumulates and can offset runtime gains.
  • ✗Simplified representation — Uses truncated time horizons and compressed trajectory encodings, which may lose important interaction details in complex scenarios.
  • ✗Limited realism — Evaluated only on grid-based MAPF benchmarks; does not account for richer domains like the hospital setting with additional constraints.
Verdict: Architecturally innovative and scientifically valuable — the conv-attention interleaving is a contribution beyond MAPF. But practical gains are contested, and deployment limitations (GPU, retraining, no adaptivity) are significant. Main value: proof-of-concept and blueprint.
